## Следующие приоритетные задачи

### 1. A/B тесты (HIGH priority)
**Что тестировать:**
- Порядок этапов: тест в начале vs в середине
- Длина интервью: 5 вопросов vs 3 вопроса
- Визуал приветствия: текст vs видео от CEO
- Rewards: без наград vs с бейджами и XP

**Реализация:**
- Добавить `ab_test_variant` в session_state
- Создать эндпоинт `/v1/ab/assign` для рандомизации
- Логировать варианты и конверсии

### 2. Альтернативные вакансии при отказе (MEDIUM)
**Идея:** При отклонении показывать похожие вакансии с более низкими требованиями

**Реализация:**
- Эндпоинт `/v1/jobs/alternatives?skills=...`
- Блок на странице rejection с рекомендациями
- "Откликнуться без теста" для Junior позиций

### 3. HR Analytics Dashboard (MEDIUM)
**Метрики:**
- Воронка конверсии по этапам
- Среднее время прохождения
- Drop-off rate по этапам
- Quality score кандидатов
- A/B тест результаты
